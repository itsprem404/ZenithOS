> **A Terminal-Based Personal Productivity Operating System built with C++17**

## ✨ Features

- 🔐 **Secure Login System** - User authentication with attempt limiting
- 📝 **Notes Manager** - Create, search, edit, delete, categorize notes
- ✅ **Task Scheduler** - Priority-based task management with due dates
- 📁 **File Organizer** - Browse, search, organize files by type
- 🛠️ **Built-in Tools** - Calculator, Calendar, System Info
- 🎨 **Beautiful Terminal UI** - Colors, animations, ASCII art

## 🚀 Quick Start

### Prerequisites
- C++17 compatible compiler (g++, clang++)
- Make (optional)

### Build & Run

```bash
# Clone repository
git clone https://github.com/yourusername/ZenithOS.git
cd ZenithOS

# Build using Make
make

# Run
./zenithos
